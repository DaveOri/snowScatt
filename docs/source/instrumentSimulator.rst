instrumentSimulator
===================

This module provides an easy implementation of common microwave remote sensing instrument forward models. The focus is rapid development and not accuracy of computation.
So far only a simplicistic radar Doppler and moments simulator has been implemented.
The module also implements a series of conveniency functions to formulate common used particle size distributions
